# 📖 בודק מקורות

**אפליקציית דסקטופ ל-Windows לבדיקה אוטומטית של הפניות בספרות תורנית.**

המשתמש מזין מסמך (`.txt` / `.docx` / `.pdf`) המכיל הפניות למקורות בסוגריים
(למשל `{בראשית א, א}` או `{שבת לא:}`). התוכנה מחלצת את ההפניות, מנרמלת אותן,
בודקת כל אחת מול מסד הנתונים המקומי של **אוצריא** (ובגיבוי מול **Sefaria.org**),
ומציגה השוואה חזותית צד-לצד בין מה שנכתב לבין הטקסט המקורי במאגר — בזמן אמת.

> נבנה ב-**Tauri 2** (ליבת Rust + WebView מערכתי). קובץ סופי קטן ומהיר,
> ללא Node ו-Electron בזמן ריצה.

---

## ✨ יכולות

- **תמיכה במגוון קבצים** — `.txt`, `.docx`, ו-`.pdf` (חילוץ טקסט).
- **חיפוש חכם** — נרמול מספרים עבריים (גימטריה), נרמול דפי גמרא
  (`לא ע״א` → `31.`), פיתוח קיצורי מסכתות (`ב״ק` → בבא קמא), ובניית וריאנטים.
- **התאמה מדורגת** — `exact` → `prefix` → חיפוש מטושטש (FTS5 אם קיים, אחרת `LIKE`).
- **הצגה בזמן אמת** — כל תוצאה מופיעה ברגע שנמצאת, עם אפשרות לעצור באמצע.
- **השוואה חזותית** — הציטוט מהמקור לצד הטקסט מהמאגר, עם הדגשת הביטוי.
- **הרחבת דף גמרא** — לש"ס בבלי, פתיחת הדף המלא (±40 שורות).
- **Sefaria Fallback** — מקורות שלא נמצאו מקומית נחפשים אוטומטית ב-Sefaria.org.
- **היסטוריה, הגדרות, וייצוא** (CSV / TXT), מגש מערכת (Tray), ועיצוב RTL מלא.

## 🏗️ ארכיטקטורה

```
src/                  ← Frontend סטטי (HTML / CSS / JS, RTL)
  ├── index.html      ← 4 עמודים: השוואה / היסטוריה / אודות / הגדרות
  ├── main.js         ← invoke + listen (IPC) — ללא לוגיקת חיפוש
  └── style.css       ← ערכת "מגילה עתיקה"
src-tauri/            ← ליבת Rust
  ├── src/main.rs     ← כל הלוגיקה: חילוץ, נרמול, התאמה, FTS5, Sefaria, דף גמרא
  ├── Cargo.toml      ← rusqlite (bundled+FTS5), tauri, reqwest, fancy-regex, ...
  ├── tauri.conf.json
  └── capabilities/default.json
```

תקשורת Frontend↔Backend היא דרך פקודות Tauri (`invoke`) ואירועים (`listen`),
לא דרך שרת HTTP. הפקודות: `compare_start`, `compare_abort`, `expand_page`, `pick_file`.
האירועים: `compare-result` (פר הפניה, בזמן אמת), `compare-done` (סיכום).

## 🚀 בנייה והרצה

ראו את **[BUILD.md](BUILD.md)** להוראות מלאות צעד-אחר-צעד.

בקצרה (Windows, לאחר התקנת Rust + C++ Build Tools + WebView2):

```powershell
cargo tauri dev      # הרצה במצב פיתוח
cargo tauri build    # installer לשחרור → src-tauri/target/release/bundle/nsis/
```

`C:/ProgramData/otzaria/books/seforim.db` (ניתן לשינוי בהגדרות).

הקוד מזהה אוטומטית את שמות העמודות — גם סכמת אוצריא (`heRef`, `bookId`,
